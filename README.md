# TQM & KPI 管理系統

這是一個整合了 **TQM (全面品質管理)** 與 **KPI (關鍵績效指標)** 的綜合管理平台。

## 系統架構

本系統採用前後端分離架構 (但在單一 HTML 檔案中實作前端)，支援多人同時連線使用。

- **前端**: React 18 (透過 Babel Standalone 編譯), Tailwind CSS (樣式), Lucide React (圖標), Recharts (圖表)
- **後端**: Node.js Express Server
- **資料庫**: JSON 檔案資料庫 (db.json)

## 功能模組

### 1. 登入與選單系統
- **使用者驗證**: 支援多使用者登入，區分管理員與一般使用者權限。
- **系統選單**: 登入後可選擇進入「TQM 品質管理系統」或「KPI 績效系統」。

### 2. TQM 品質管理系統
- **異常紀錄**: 新增、修改、刪除品質異常紀錄。
- **資料匯入**: 支援 Excel (CSV) 批量匯入舊資料 (自動處理 Big5 編碼與日期格式)。
- **統計分析**: 提供年度趨勢圖、部門案件統計、異常類型排行。
- **報表匯出**: 支援匯出 CSV 報表。

### 3. KPI 績效系統
- **績效儀表板**: 視覺化呈現達成率、評核人數、異常項目。
- **個人分析**: 針對特定員工進行年度績效趨勢分析。
- **詳細報表**: 列表式查看所有員工績效，支援篩選與搜尋。
- **資料填寫**: 線上填寫績效指標與評核結果。

## 安裝與執行

### 快速啟動 (推薦)
直接執行根目錄的 `啟動伺服器.bat`，系統會自動：
1. 啟動 Node.js 伺服器
2. 等待 3 秒讓伺服器完全啟動
3. 自動開啟 TQM.html 網頁

### 手動啟動
#### 1. 啟動伺服器 (Server)
在 server 資料夾中開啟終端機 (Terminal)，執行：
```bash
node server.js
```
伺服器將在 http://localhost:3002 啟動。

#### 2. 開啟應用程式 (Client)
直接使用瀏覽器 (建議 Chrome/Edge) 開啟 server/TQM.html 檔案。

## 資料庫備份與還原
- **備份**: 在 TQM 系統側邊欄點擊「匯出資料庫 (備份)」，可下載完整 JSON 檔。
- **還原**: 在登入畫面點擊「載入資料庫」，選擇備份的 JSON 檔即可還原。

## 開發說明
- 前端程式碼位於 TQM.html 中。
- 後端 API 定義於 TQM_Server/server.js。
- 資料儲存於 TQM_Server/db.json。
